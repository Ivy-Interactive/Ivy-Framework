# Image

<Ingress>
Display images with automatic loading, responsive sizing, and proper accessibility features for rich visual content.
</Ingress>

The `Image` widget displays images in your app.

```csharp demo-below 
Layout.Horizontal()
    | new Image("https://api.images.cat/150/150")
```
